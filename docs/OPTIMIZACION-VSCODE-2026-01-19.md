# Optimización de VS Code aplicada

- Se agregó la línea:
  "files.maxMemoryForLargeFilesMB": 4096
  en el archivo settings.json de usuario para permitir abrir archivos grandes sin trabas.
- Recomendaciones adicionales: cerrar extensiones innecesarias, reiniciar VS Code periódicamente, y dividir el trabajo en carpetas pequeñas.
